# Desktop Shortcut for Gnome Shell
Light gnome extension that adds an item to the gnome shell icons menu leading to the .desktop file

## Installation

### How to manually install the extension

```
git clone git@github.com:nattesh/open-desktop-location.git \
	~/.local/share/gnome-shell/extensions/open-desktop-location@nattesh
```

You will need to restart the Gnome Shell (for manual install):
Logout and login again.
